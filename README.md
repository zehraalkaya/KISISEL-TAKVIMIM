# Kişisel Takvim Uygulaması (C)

C dili ile geliştirilmiş, kullanıcıların tarih bazlı etkinliklerini yönetebildiği konsol tabanlı bir uygulamadır.

## 🛠 Özellikler
- **Etkinlik Yönetimi:** Dinamik olarak yeni etkinlik ekleme ve silme.
- **Listeleme:** Kayıtlı tüm etkinlikleri tarih sırasına göre görüntüleme.
- **Menü Yapısı:** Kullanıcı dostu ve yönlendirmeli kontrol paneli

## 💻 Teknik Detaylar
- **Dil:** C
- **Veri Yapıları:** Verilerin organize edilmesi için `Struct` yapısı kullanılmıştır
- **Kontrol Akışı:** Döngüler ve karar yapıları ile optimize edilmiş algoritma mantığı
